Container

作為一個最大寬度的限制，有效將使用者視線留在最高停留的區域：1366px (22.98% - 2020 年數據)

推薦大家閱讀一下這篇科普文章： 【 DESIGNRUSH NEWS: Website Dimensions 】


HTML

<div>
    <div class="container">{{被控制寬度的內容物}}</div>
</div>

SCSS / CSS


    .container {
        width: 100%;
        max-width: 1366px; /*將視線集中*/
        margin: 0 auto;
    }
