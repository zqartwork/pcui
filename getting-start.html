開始使用說明書

請詳閱以下介紹與脈絡，並且結合你平常的使用習慣中。

下載或使用CDN說明

  • 下載： 本地端開發時，為了避免網路問題而造成顯示問題，會使用下載的方式將套件下載並且放入專案資料夾中。缺點是第一次載入時間較長，但是之後載入速度幾乎接近0秒。
  • CDN： 節省下載資源，進入Production之後可以有效減少流量以及使用者第一次載入時間，之後載入時間相當。現階段企業廣用此方式載入套件。

常用外接套件

  1. meyer-reset.css 梅爾重置

    用以去除基礎style，減少每次設定系統style的功夫。

  2. animate.css + wow.js 滾輪進場動畫

    簡易 Scroll Animation，網站常用一次性進場動畫。

    安裝文件以及使用辦法：https://wowjs.uk/docs.html

  3. OwlCarousel 貓頭鷹輪播套件

    輕量級輪播套件，RWD、跨瀏覽器以及手勢滑動，Codepen 上也有很多改寫 Source 分享。

    安裝文件以及使用辦法：https://owlcarousel2.github.io/OwlCarousel2/

  4. FontAwesome.js 免費開源圖示庫

    此資源提供 Source Code, svg 圖檔，現今常用軟體縮圖 icon 也有提供。

    圖示搜尋表：https://fontawesome.com/icons?d=gallery&p=2
